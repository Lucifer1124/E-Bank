E-Bank
Orchestrated end-to-end development of AI-Powered Full-Stack Banking Platform using React, Spring Boot, PostgreSQL, FastAPI, Docker
Built a secure banking platform featuring JWT-based authentication, role-based admin/user dashboards, account creation, KYC management, fund transfers, transaction history, Razorpay payment integration, loan eligibility and repayment workflows, fraud detection, and AI chatbot support. Developed Spring Boot REST APIs with Spring Security, JPA/Hibernate, PostgreSQL, SMTP-based OTP password reset, and Gemini API integration. Created FastAPI ML services for fraud detection and loan scoring using XGBoost, scikit-learn, pandas, and SMOTE. Containerized the full system with Docker Compose, served the frontend with Nginx, and deployed the application on Render.

Tech Stack: Java 21, Spring Boot 3.5, Spring Security, Spring Data JPA, Hibernate, JWT, BCrypt, JavaMail SMTP, Razorpay, Gemini API, PostgreSQL, React 19, React Router, Axios, Bootstrap 5, Framer Motion, FastAPI, Uvicorn, Pandas, NumPy, Scikit-learn, XGBoost, SMOTE, Docker, Docker Compose, Nginx, Render.
